# Getting Started with Create React App


### 快速开始
```sh
npm install # OR yarn
npm start   # OR yarn start
```

[http://localhost:3000](http://localhost:3000) to view it 

### `npm test`
### `npm run build`
### `npm run eject`

**Note: this is a one-way operation. Once you `eject`, you can't go back!**
